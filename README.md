# Scattered Invoice Threat Hunt Page

A static GitHub Pages-ready threat hunting report for the Scattered Invoice BEC/MFA fatigue investigation.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, and this `README.md` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. Your page will be available at your GitHub Pages URL.

## Customize

- Replace `Your Name` in `index.html` with your analyst name.
- Update the Case ID if needed.
- Add screenshots by placing images in an `assets/` folder and embedding them in the relevant sections.
